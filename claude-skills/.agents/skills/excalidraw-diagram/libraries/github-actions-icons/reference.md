# github-actions Reference

This directory contains 4 icons extracted from `github-actions.excalidrawlib`.

## Available Icons

| Icon Name | Filename |
|-----------|----------|
| Cache | `icons/Cache.json` |
| GitHub | `icons/GitHub.json` |
| GitHub Actions | `icons/GitHub-Actions.json` |
| Metrics | `icons/Metrics.json` |

## Usage

Each icon JSON file contains the complete `elements` array needed to render that icon in Excalidraw.
You can copy the elements from these files into your Excalidraw diagrams.
